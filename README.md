# SYNERGY Denver Orientation Video Assets

Production assets for the 15-scene caregiver orientation video.

## Layout

- Canvas: 1920x1080
- Left 480px: SYNERGY navy zone with branded wordmark for presenter (Esmeralda in SYNERGY red polo)
- Right 1440px: original SYNERGY slide content, scaled with margin, soft drop shadow
- 4px red separator bar between zones
- "SYNERGY HomeCare / of Denver / CAREGIVER ORIENTATION" header in avatar zone
- Scene counter "N / 15" at bottom-left
- "Brainstorm AGI LLC" footer

## Files

- composites/scene-NN.png - pre-built 1920x1080 backgrounds for each scene, including navy zone, slide, branding, and shadow. Just drop presenter in.
- scripts.txt - scene-by-scene spoken scripts
- scripts.json - same data in JSON format
- final/ - final rendered MP4 goes here after production

## Public fetch URLs

https://raw.githubusercontent.com/HCJefe/synergy-denver-orientation/main/composites/scene-01.png through scene-15.png
